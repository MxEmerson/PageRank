# PageRank

PageRank算法的Python标准库实现

## 目录说明

```shell
│  main.py          # 入口
│  README.md
├─bin
│  │  pagerank.py   # pagerank算法
│  └─ __init__.py
├─docs
│  └─ PageRank.md
└─lib
   │  generator.py  # 生成矩阵
   │  matrix.py     # 列表表示的矩阵相乘
   └─ __init__.py
```

## 运行环境

- Linux / Windows
- Python3.7+, sqlite3

## 运行

```shell
> python main.py
```
